# BookingSystem (Система онлайн-записи)
Десктопное приложение на C# для автоматизации записи клиентов и управления расписанием мастеров. Позволяет клиентам записываться на услуги, а мастерам эффективно управлять своим рабочим графиком, услугами и статистикой.

## О проекте

### Почему был создан этот продукт?
Проект разработан в рамках учебной практики как практическое решение для малого бизнеса в сфере услуг (салоны красоты, сервисные центры, медицинские центры). Он сочетает в себе реальную бизнес-задачу с возможностью применить знания по разработке полноценного клиент-серверного приложения на C#.

### Какую проблему он решает?
Приложение автоматизирует процесс записи клиентов, устраняя необходимость ведения бумажных журналов или использования неудобных таблиц. Оно решает проблему "двойного бронирования", контролирует рабочее время мастеров и предоставляет аналитику для оптимизации бизнес-процессов.

### В чём его ключевые преимущества?
* Простота использования — интуитивно понятный интерфейс для клиентов и мастеров
* Автоматизация процессов — проверка доступности времени, напоминания, статистика
* Надёжность данных — транзакционная работа с базой данных MySQL
* Гибкость настройки — мастера могут полностью настраивать свои услуги и расписание
* Локальное хранение — все данные хранятся на вашем сервере или компьютере

## Быстрая установка и запуск

Вы можете запустить приложение за несколько минут, используя готовые файлы:

**1. Установите необходимое ПО:**
* Установите .NET Framework 4.8 по [этой ссылке](https://dotnet.microsoft.com/download/dotnet-framework/net48)
* Установите MySQL Server 8.0 по [этой ссылке](https://dev.mysql.com/downloads/mysql/)
* Во время установки запомните пароль пользователя root
* Выберите стандартные настройки
* Установите .NET Desktop Runtime, если требуется

**2. Настройка базы данных:**
* Скачайте готовый скрипт базы данных [Script_BookingSystem.sql](https://github.com/Lision0k/BookingSystem/blob/main/DataBase/Script_BookingSystem.sql)
* Откройте командную строку MySQL (или обычную командную строку Windows)
* Выполните команду для импорта данных, указав правильный путь к скачанному файлу:

```bash
mysql -u root -p < "C:\Путь\к\папке\Script_BookingSystem.sql"
```

* Вас попросят ввести пароль пользователя root MySQL
* После успешного выполнения вы увидите сообщение об импорте данных 

**3. Запустите приложение:**
* Скачайте готовый исполняемый файл [BookingSystem.exe](https://github.com/Lision0k/BookingSystem/blob/main/BookingSystem.exe)
* Запустите приложение одним из способов: Двойным кликом по BookingSystem.exe / Через командную строку:
```bash
"C:\Путь\к\папке\BookingSystem.exe"
```
> Важно: Перед запуском убедитесь, что сервис MySQL Server запущен в вашей системе.

## Возможности

### Для мастеров:
* Умное расписание — календарный вид, контроль пересечений записей
* Управление услугами — создание, редактирование, деактивация
* Статистика — загрузка, доходы, популярные услуги, загруженные дни
* Профессиональный профиль — фото, специализация, опыт, описание
* Управление клиентами — просмотр записей, контакты клиентов
* Финансовая аналитика — предполагаемый доход, средняя загрузка

### Для клиентов:
* Онлайн-запись — выбор мастера, услуги и удобного времени
* Просмотр расписания — календарь с занятыми и свободными слотами
* Управление записями — просмотр, отмена (за 2 часа до начала)
* Поиск мастеров — фильтрация по услугам, просмотр профилей с фото
* История посещений — все выполненные и предстоящие записи
* Личный профиль — редактирование данных, смена пароля

## Использование

### Начало работы:
1. Запустите приложение и выберите "Регистрация" если вы новый пользователь
2. Выберите роль — Клиент или Мастер
3. Заполните профиль — укажите ФИО, телефон, специализацию (для мастеров)

### Для мастеров:
* Войдите в систему под учетной записью мастера
* Добавьте свои услуги во вкладке "Мои услуги"
* Настройте профиль — загрузите фото, укажите опыт и описание
* Просматривайте расписание на день/месяц
* Анализируйте статистику во вкладке "Расписание"

### Для клиентов:
* Войдите в систему под своей учетной записью
* Нажмите "Новая запись" на главном экране
* Выберите услугу → мастера → дату → доступное время
* Просматривайте свои записи в разделе "Мои записи"
* Отменяйте записи при необходимости (минимум за 2 часа)

## Разработка и тестирование
Проект был разработан с использованием JavaFX для создания пользовательского интерфейса и паттерна MVC для организации кода. Доступ к данным реализован через отдельные DAO-классы (Data Access Object).

### Тестирование
В проекте проведено комплексное тестирование, включающее:
* Модульное тестирование: Проверка корректности работы отдельных компонентов, таких как расчёт баланса, валидация данных, работа фильтров.
* Интеграционное тестирование: Проверка взаимодействия между модулями (например, добавление транзакции → обновление баланса и бюджета). Тестирование проводилось на основе управляющего графа программы, охватывая основные пользовательские сценарии
